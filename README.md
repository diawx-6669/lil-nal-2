# 🌐 FlowMind — Интеллектуальная панель управления

**FlowMind** — это современная корпоративная экосистема для мониторинга бизнес-процессов, транзакций и состояния инфраструктуры в реальном времени. Система отличается строгим, профессиональным дизайном в стиле "dark mode" и высокой плотностью представления данных.

---

## 🔐 Модуль авторизации (Auth)
Система использует многоэтапную регистрацию для глубокого профилирования пользователя:
* **Безопасность:** Проверка пароля (минимум 8 символов).
* **Корпоративный профиль:** Сбор данных о компании, типе деятельности и объемах транзакций.
* **Сессионность:** Использование `localStorage` для хранения данных сессии, что позволяет мгновенно восстанавливать доступ к дашборду.

## 📊 Модуль мониторинга (Dashboard)
Дашборд является центральным хабом FlowMind и включает в себя:

* **Индикация транзакций:** Секция "ПУЛЬС ТРАНЗАКЦИЙ" (Pulse Feed), отображающая активность в реальном времени.
* **Финансовая аналитика:** Табличное представление активов с использованием цветовой индикации (теги валют, прогресс-бары, статусы "Норма" и "Задержка").
* **Мониторинг здоровья инфраструктуры:** Визуализация состояния серверов и узлов сети.
* **UX/UI особенности:**
    * Использование шрифтов *Space Mono* (для моноширинных данных) и *Outfit* (для основного текста).
    * Адаптивная архитектура, готовая к работе в браузерах нового поколения.
    * Интеграция иконок через *Tabler Icons*.

---

## 🛠 Технологический стек

* **Frontend:** Чистый HTML5, CSS3, JavaScript (Vanilla).
* **Типографика:** Google Fonts (Space Mono, Outfit).
* **Графические ресурсы:** Tabler Icons.
* **Логика состояния:** Управление данными через `localStorage` и кастомные JS-скрипты.
